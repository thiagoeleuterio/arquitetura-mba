# Meu Projeto Worktree

Este é o repositório principal do projeto, que contém a aplicação principal e suas funcionalidades.

## Estrutura do Projeto

- **main**: Contém a aplicação principal.
  - **src**: Código-fonte da aplicação principal.
  - **package.json**: Configuração do npm para a aplicação principal.
  - **tsconfig.json**: Configuração do TypeScript para a aplicação principal.
  - **README.md**: Documentação da aplicação principal.

- **feature-login**: Contém a funcionalidade de login.
  - **src**: Código-fonte da aplicação de login.
  - **package.json**: Configuração do npm para a aplicação de login.
  - **tsconfig.json**: Configuração do TypeScript para a aplicação de login.
  - **README.md**: Documentação da aplicação de login.

- **hotfix-api**: Contém a funcionalidade de correção de API.
  - **src**: Código-fonte da aplicação de correção de API.
  - **package.json**: Configuração do npm para a aplicação de correção de API.
  - **tsconfig.json**: Configuração do TypeScript para a aplicação de correção de API.
  - **README.md**: Documentação da aplicação de correção de API.

## Como Executar

Para executar a aplicação principal, navegue até o diretório `main` e execute os seguintes comandos:

```bash
npm install
npm start
```
